 # SELL THAT OUT 
 > USED TO BUY AND SELL OLD PRODUCTS

  [Site Link: https://sell-that-out.herokuapp.com/
](https://sell-that-out.herokuapp.com/

## Setup
```
npm init
npm install 
npm start
```
Open in Browser ```http://localhost:5555/```
